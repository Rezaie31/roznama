# روزنما — اپلیکیشن ثبت کارهای روزانه

فرانت‌اند رو گیت‌هاب پیجز هاست می‌کنه، بک‌اند و دیتابیس هردو گوگل شیت + گوگل اپس اسکریپت هستن (رایگان، بدون نیاز به سرور).

## مرحله ۱: ساخت گوگل شیت و بک‌اند

1. یه [Google Sheet](https://sheets.new) جدید بساز (اسمش مهم نیست، مثلاً "Roznama DB").
2. از منو بالا: **Extensions → Apps Script**
3. کل محتوای فایل `Code.gs` رو کپی کن و به‌جای کد پیش‌فرض تو ادیتور اپس اسکریپت بچسبون.
4. بالای صفحه، دکمه **Save** (آیکون فلاپی) رو بزن.
5. دکمه آبی **Deploy → New deployment** رو بزن.
6. کنار "Select type"، روی چرخ‌دنده کلیک کن و **Web app** رو انتخاب کن.
7. تنظیمات:
   - **Execute as:** Me
   - **Who has access:** Anyone
8. **Deploy** رو بزن. اولین بار ازت می‌خواد دسترسی بدی (Authorize access) — قبول کن.
9. یه لینک به شکل زیر بهت میده، کپیش کن:
   ```
   https://script.google.com/macros/s/XXXXXXXXXXXX/exec
   ```

> اگه بعداً کد `Code.gs` رو عوض کردی، باید دوباره **Deploy → Manage deployments → ویرایش (مداد) → Version: New version → Deploy** بزنی تا تغییرات اعمال بشه.

## مرحله ۲: وصل کردن فرانت‌اند به بک‌اند

فایل `config.js` رو باز کن و لینکی که از مرحله قبل کپی کردی رو جایگزین کن:

```js
const APPS_SCRIPT_URL = "https://script.google.com/macros/s/XXXXXXXXXXXX/exec";
```

## مرحله ۳: آپلود روی گیت‌هاب پیجز

1. یه ریپازیتوری جدید تو گیت‌هابت بساز (مثلاً `roznama`).
2. همه فایل‌های این پوشه (`index.html`, `dashboard.html`, `style.css`, `api.js`, `config.js`, `landing.js`, `dashboard.js`) رو آپلود کن. **`Code.gs` نیازی نیست آپلود بشه** (اون فقط تو گوگل اپس اسکریپت می‌مونه).
3. برو تو تنظیمات ریپازیتوری: **Settings → Pages**
4. تحت "Branch"، شاخه `main` رو انتخاب کن و Save بزن.
5. بعد چند دقیقه، سایتت روی یه لینک شبیه این بالا میاد:
   ```
   https://username.github.io/roznama/
   ```

## نکته امنیتی

رمز عبورها قبل از ارسال با SHA-256 هش می‌شن (نه متن ساده)، ولی این یه سیستم احراز هویت درجه‌یک و امن در حد بانکی نیست — برای یه پروژه شخصی یا نمونه‌کار (پورتفولیو) کاملاً مناسبه، ولی برای دیتای حساس واقعی توصیه نمی‌شه.

## ساختار دیتا در گوگل شیت

اپس اسکریپت خودش موقع اولین اجرا این ۴ شیت رو می‌سازه:
- **Users** — نام کاربری، هش رمز عبور، تاریخ ثبت‌نام
- **Tasks** — کارهای تعریف‌شده هر کاربر
- **Logs** — ثبت روزانه‌ی انجام/عدم‌انجام هر کار
- **Notes** — یادداشت‌های روزانه هر کاربر

می‌تونی مستقیم تو خود گوگل شیت این دیتا رو ببینی و حتی دستی ویرایش کنی — دقیقاً همون "ابرکنترل‌گر" که خواستی.
